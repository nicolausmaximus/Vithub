
# Vithub
Created a Student portal for VITians to enable easy, safe and free file sharing.

## Deployment

To clone this project, run

```bash
  git@github.com:nicolausmaximus/Vithub.git
```

To install all the dependencies
```bash
  npm i
```

To run this project

```bash
  node/nodemon app.js
```
